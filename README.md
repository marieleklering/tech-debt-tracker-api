## Run locally
```bash
# Clone the repo
git clone https://github.com/marieleklering/tech-debt-tracker-api.git

# Open the folder
cd tech-debt-tracker-api

# Install grip (one-time)
pip install grip

# Start the server
grip
```

Then open http://localhost:6419
